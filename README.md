# ComedorConectado 🍽️

Sistema inteligente para la gestión y optimización de comedores comunitarios o institucionales.

## 🚀 Estructura del Proyecto

El proyecto está organizado en los siguientes directorios:

- **`frontend/`**: Aplicación cliente (React/Vite).
- **`backend/`**: Servidor y API (Node.js/Python).
- **`docs/`**: Documentación técnica, diagramas y especificaciones.

## 🛠️ Tecnologías Sugeridas

- **Frontend**: React, CSS Moderno.
- **Backend**: Por definir.
- **Base de Datos**: Por definir.
